# 光脉智瞳

基于分布式光纤传感与深度学习的城市生命线智能监测系统。

## 技术路线

光纤原始信号 → 预处理(去噪/FFT) → 特征提取(频域能量/峭度/峰值频率) → 轻量化分类器(SVM/1D-CNN) → 三级预警输出 → GIS可视化

## 核心指标

- 单套监测距离：≥25 km
- 定位精度：≤5 m
- 事件识别准确率：≥93%
- 误报率：≤5%
- 响应时间：≤5 s
- 国产化率：100%
